# IndoTextNormalize
Daftar kamus yang dirancang khusus untuk membersihkan, menormalisasi, dan meningkatkan kualitas korpus teks bahasa Indonesia untuk keperluan NLP Bahasa Indonesia, seperti:
- Kata slang, contohnya: `tdk` menjadi `tidak`
- Kata tidak baku, contohnya: `belom` menjadi `belum`
- Pemisahan kata tanpa spasi, contohnya: `lebihmudah` menjadi `lebih mudah`
- Penanganan Perpanjangan Huruf Ekspresif, contohnya: `baguss` menjadi `bagus`
